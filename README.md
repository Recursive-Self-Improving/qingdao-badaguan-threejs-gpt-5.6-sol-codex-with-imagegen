# 八大關 · 海風穿過林蔭

一個以 Three.js 建構的青島八大關互動虛擬景觀。場景以林蔭緩坡、獨立花園別墅、花崗岩牆、紅瓦屋頂與太平灣海岸為核心，重建雨後夏末的海濱街區氛圍。

## 啟動

```bash
npm install
npm run dev
```

正式建置：

```bash
npm run build
npm run preview
```

## 操作

- `W A S D` 或方向鍵移動
- 滑鼠移動旋轉視角
- `Shift` 快走
- 靠近景點時按 `E` 閱讀介紹
- `Esc` 釋放滑鼠
- 觸控裝置使用左側搖桿移動、右半畫面拖曳環顧

右上角可切換「海霧金時／清晨薄霧／暮色藍調」，並開關程序化海風與浪聲。

## 場景說明

這是一段依據八大關真實空間特徵所做的氛圍重建，不是精確測繪或單棟建築的數位複刻。封面影像為專案生成的原創素材；三維部分均由程序化幾何、材質與著色器構成。
